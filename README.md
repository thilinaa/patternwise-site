# Patternwise — website

Public pages for **Patternwise**, a private, local-first wellness journal for iPhone that helps
you notice your own health patterns — migraine, reflux, sleep, food and the weather around you,
tracked together on your device.

- `index.html` — home
- `support.html` — Support &amp; FAQ
- `privacy-policy.html` — Privacy Policy

Plain HTML, no build step, served via GitHub Pages.
